## Step 1: Instalar dependencias

Primero necesitas instalar las dependencias

```bash
# using npm
npm install

# OR using Yarn
yarn install
```

Si usas iOS, debes correr el siguiente comando

```bash
bundle install

npx pod-install ios
```

## Step 2: Correr aplicación

Primero debes encender el Metro, con el siguiente comando:

```bash
# using npm
npm start

# OR using Yarn
yarn start
```

Ahora para levantar la aplicación debes hacer lo siguiente:

### Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
